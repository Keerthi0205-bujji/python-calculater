# 🧮 Python Calculator

A simple, beginner-friendly calculator built with Python.  
This project demonstrates basic arithmetic operations and serves as a foundation for learning Python programming.

---

## ✨ Features
- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division (with error handling for division by zero)
- 🔢 User-friendly command-line interface
